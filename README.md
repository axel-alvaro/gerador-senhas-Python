# gerador-senhas-Python
Gerador de senhas desenvolvido em Python que cria senhas aleatórias com comprimento personalizado. O projeto utiliza manipulação de strings, funções e bibliotecas nativas do Python para gerar combinações seguras de caracteres, números e símbolos


# Gerador de Senhas em Python

Projeto desenvolvido para gerar senhas aleatórias de forma simples e rápida.

## Funcionalidades

- Definir tamanho da senha
- Gerar caracteres aleatórios
- Suporte a letras, números e símbolos

## Tecnologias

- Python
- Random
- String

## Como executar

python gerador_senhas.py
